# 🏦 Bankist Website

A modern, responsive banking landing page built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project demonstrates advanced DOM manipulation, smooth user interactions, and performance optimization techniques.

## 🌐 Live Demo

🔗 **Live Website:**  https://bankiome.netlify.app/

>
>
> 
## ✨ Features

- 📱 Fully responsive design
- 🎯 Smooth scrolling navigation
- 📌 Sticky navigation bar
- 🪟 Modal window
- 📑 Tabbed component
- 🖼️ Lazy loading images
- 🎞️ Image slider/carousel
- 👀 Reveal sections on scroll
- ⚡ Optimized using Intersection Observer API
- ⌨️ Keyboard navigation support

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6+)
- DOM Manipulation
- Intersection Observer API

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/bankist-website.git
```

Navigate to the project folder:

```bash
cd bankist-website
```

Open `index.html` in your browser.

Or use VS Code Live Server:

```bash
Right Click → Open with Live Server
```
