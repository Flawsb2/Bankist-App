# 🏦 Bankist App

A modern banking application built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. The app simulates core online banking operations such as logging in, transferring money, requesting loans, closing accounts, and viewing transaction history.

## 🌐 Live Demo

🔗 **Live Website:** https://bankiome.netlify.app/

---


---

## ✨ Features

- 🔐 Secure user login
- 💸 Transfer money between accounts
- 💰 Request loans
- 🗑️ Close bank account
- 📊 View account balance and transaction history
- 📈 Automatic interest calculation
- 🔄 Sort transactions
- ⏳ Automatic logout timer for security
- 📅 Display formatted transaction dates
- 💱 International currency formatting
- 📱 Responsive user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- DOM Manipulation
- Internationalization API (`Intl`)
